# Adoption Guide

> Placeholder. The adoption guide lands with v0.5.0 — see [ROADMAP.md](../ROADMAP.md).

When v0.5.0 ships, this document will be a step-by-step walkthrough:

1. Install `uv` and run `uv tool install company-brain`.
2. Pick a profile (`medical-device` or `default`).
3. Scaffold a new vault with `cb vault-architect`.
4. Run a 30-minute `vision` intake session.
5. Atomize an existing document (PDF, Word, transcript, or screenshot).
6. Generate an MRD.
7. View the vault graph in the browser.
