# Citation: Accelerate (Forsgren, Humble, Kim)

## Summary

The Accelerate book defined the four DORA metrics (lead time, deployment frequency, change failure rate, time to restore) that became the industry vocabulary.

## How we use it

- Onboarding doc references it as the canonical primer on engineering productivity metrics.
- Our `concept-cycle-time` node derives from the Accelerate definition.
- When customers ask "why these four metrics?" the right answer is "because Accelerate showed they correlate with organizational performance — we did not invent them."
